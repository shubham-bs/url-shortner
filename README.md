# URL Shortener

A Spring Boot based URL shortening service built to explore backend application design, layered architecture, and production-style development practices.

## Features

- Create short URLs
- List existing URLs
- URL redirection (in progress)

## Tech Stack

- Java
- Spring Boot
- Thymeleaf
- Spring Data JPA
- PostgreSQL

## Project Structure

- `ARCHITECTURE.md` — Current system design and request flows
- `JOURNAL.md` — Project evolution and major milestones

## Roadmap

- URL redirection
- Authentication
- User-owned URLs
- Custom aliases
- Expiration support
- Analytics
- QR code generation
